Juego Web - Adivina el personaje de Hunter x Hunter

🎮 Descripción:
Este juego web intenta adivinar personajes del anime Hunter x Hunter mediante preguntas. Si no adivina, puedes enseñarle nuevos personajes.

📦 Requisitos:
- Python 3.7+
- Flask

🧪 Instalación (Windows):
1. Ejecuta `setup_venv.bat`
2. Ejecuta el juego con: python app.py
3. Abre http://localhost:5000

🧪 Instalación (Linux/macOS):
1. Ejecuta: chmod +x setup_venv.sh
2. Luego: ./setup_venv.sh
3. Ejecuta el juego con: python app.py

📁 Archivos principales:
- app.py: Lógica del juego
- personajes.json: Base de datos de personajes
- requirements.txt: Dependencias
- setup_venv.bat / setup_venv.sh: Script de instalación automática

📝 Cómo agregar personajes:
El juego te preguntará si deseas enseñar nuevos personajes si no los reconoce.
